The Clinical Pipeline

DermAI bridges the gap between patient skin conditions and clinical evaluations. We have built a fully software-driven platform that integrates computer vision models (hosted on Hugging Face) with conversational LLM routers (Google Gemini APIs) to evaluate skin textures and return instant condition categorizations.

To establish a smooth patient-doctor workflow, the application implements custom dashboards for both roles, online video chat modules for remote consultation, secure Magic-SDK passwordless logins, and automatic Razorpay subscription billing controls.

04 / Secure Access

Passwordless Authentication

Provides secure, fast, and frictionless user access using Magic-SDK. Eliminates traditional password tables and threat vectors, generating secure cryptographic tokens verified by server APIs to authorize sessions.

06 / Engineering Details

Engineering Challenges & Resolutions

1

Patient Session Data Security

**Challenge**: Storing patient diagnostic history and macro images within shared databases introduces compliance risks if data tables are exposed.

**Resolution**: Integrated a strict tokenized anonymization schema in MongoDB. Images and patient identification credentials are coded with isolated token hashes, decryptable only through temporary frontend verification keys.

2

Telehealth Video Call Stability

**Challenge**: Negotiating low-latency peer connections across hospital firewalls and poor consumer networks causes video drops and high packet loss.

**Resolution**: Integrated reliable STUN/TURN server gateways in connection routes. Tunnels automatically scale video qualities down to keep voice logs active during severe bandwidth restrictions.

3

LLM Diagnosis Guidance Limits

**Challenge**: Gemini recommendation outputs can present over-confident clinical conclusions, exceeding basic triage assistance scopes.

**Resolution**: Implemented strict prompt context layers. The server validates Gemini prompts, appending mandatory legal disclaimers and restricting output recommendations to verified skincare products and practitioner referral guidelines.
