The Geotechnical Challenge

Subsurface geology is notoriously complex to map and predict. Hydrologists and engineering teams must synthesize multiple uncoordinated data feeds—physical logging records, structural fault maps, and academic manual citations—to locate water-bearing aquifers and guide extraction planning. Errors in stratigraphy interpretation lead to high structural failure rates and wasted resources.

The GreenBore AI Platform

GreenBore AI is designed as a software-only engineering platform to unify subsurface analysis. By taking simulated numeric sensor logs (Gamma-Ray, Resistivity, Sonic velocity) as inputs, the system models strata beds in a 3D block representation via WebGL.

Engineers run localized machine learning classifiers (RandomForest and XGBoost) to predict lithology labels in real-time, while an integrated Gemini-powered Decision Co-Pilot chat acts as a semantic query interface over dense vector databases of geological manuals.

01 / Visual Computing

Interactive 3D Subsurface Column

Implemented a high-performance WebGL geological column model using Three.js and React Three Fiber. The engine models multi-layer stratigraphy blocks, showing simulated geological strata beds, borehole path trajectories, fracture plane offsets, and water-bearing aquifer layers.

CAD-Style Inspector Features:

  • Vertical Layer Explosion Slider: Extends vertical separation distances between strata beds to inspect geological boundaries clearly.
  • Isolate Structural Items: Quickly toggle visibility of aquifers, fracture beds, or wellbore paths.
  • Opacity Controllers: Adjusts transparency of strata beds to highlight underlying structural models.

02 / Petrophysical Plotting

Petrophysical Multi-Track Log Charts

Designed viewport-scrollable high-precision SVG graphs rendering multiple geological log indicators. These tracks include Gamma-Ray (GR) for shale content, Resistivity (RES) for hydrocarbon/fluid presence, Porosity (NPHI), Density (RHOB), and Sonic velocity (DT).

Interactive Depth Synced Marker:

An interactive depth indicator tracks cursor position across SVG log charts, dynamically rotating the 3D subsurface column and passing highlighted depth values to the co-pilot chat context window for real-time analysis.

03 / Predictive Analytics

ML Model Registry & Training Console

Features an interactive model training workspace. Engineers configure parameters (estimators, learning rate, depth limits) to train local RandomForest and XGBoost classification models on logged dataset vectors.

Registry Features:

  • Local Iteration Catalog: Lists model runs with verification statistics (Accuracy, Precision, Recall, F1).
  • Production Promotion Tag: Instantly flags a trained model as the active backend predictor layer.
  • Performance Metrics: Graphs training durations and parameter sensitivity curves.

04 / Conversational AI

Decision Co-Pilot (Gemini RAG)

Built a conversational chatbot query interface dedicated to geotechnical manual searches. The frontend logs inquiries and passes them to an async FastAPI backend route, linking live sensor prediction values with vector representations of local manuals.

RAG Infrastructure specs:

  • Dense Vectors: Custom 768-dimension embeddings generated using Google Gemini Embedding-2 model.
  • RAG Pipeline: Integrates `gemini-flash-latest` with pgvector semantic similarity checks and a robust local-rules safety fallback.
  • Rich Citation Outputs: Returns clean Markdown logs featuring literal reference annotations and aligned ASCII dataset grids.

06 / Engineering Details

Engineering Challenges & Resolutions

1

WebGL Optimization for Large Strata

Challenge: Rendering complex multi-layer stratigraphy volumes with detailed mesh borders causes frame rates to drop below 30 FPS on low-power client computers.

Resolution: Implemented instanced geometries and custom vertex shaders in React Three Fiber, bypassing CPU bottlenecks and maintaining 60 FPS viewport rendering performance.

2

Vector Search Precision Fallbacks

Challenge: General embeddings queries to Google Gemini return non-specific text logs when answering complex, regional geological strata questions.

Resolution: Configured a local similarity weighting script. The pgvector search outputs are combined with localized metadata filters (depth, basin IDs) to force context alignments before calling prompt completion APIs.

3

Real-Time ML In-Browser Prediction

Challenge: Passing continuous petrophysical logging rows to server routes for classification introduces request-response latencies exceeding 400ms.

Resolution: Set up pre-compiled Scikit-Learn pipelines. Trained model coefficients are exported via JSON/Joblib and loaded directly on lightweight client threads to evaluate logs in real-time under 5ms.
